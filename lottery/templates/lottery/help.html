User Favorites

  • Setting User Favorites

    • Setting User Favorites – Navigate to the Latest Results option in the lefthand pane or Sidebar. Select this option. Here you can add your most commonly used Lottery games or games that you would like to track. Simple Click the “Plus” button under the “My Favorites” heading. From here the My Favorites dialog appears and you can add your lottery games. First select a Country, then a State or Provence. Select the game that you wish to add to your Favorites then click the Add Game button. It’s that easy!

  • Topic 3

    • To Remove a lottery game from your Favorites, Select the Red X next to the selected game.

Latest Results

  • Using Latest Results

    • After you have added your favorite lottery games to track you will use the Latest Results to see the results for the latest draw, time until the next drawing and the estimated jackpot. The estimated jackpot will only appear for certain games and it does not appear for all games.

  • Wheeling Picks

    • Wheeling lottery picks have been around for a very long time and does offer some significant advantages. Essentially, you are playing a few, several or all combinations of a given set of numbers. For example, if you were to select 6 numbers in a 6/53 game then you would get 1 ticket. If the 6 numbers you selected were all drawn then you are guaranteed a winning ticket. The more numbers that you select for the wheel, creates more combinations. There are many types of wheeling systems available like Full Wheels, Balanced Wheels, Abbreviated Wheels and so on. They all Guarantee a winning lottery prize! Careful about the Guarantee, it only applies if your numbers played are drawn, so it’s not really a guarantee.
